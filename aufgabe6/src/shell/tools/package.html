Dieses Package bietet Vorlagen für häufig verwendete Befehle, die in der Shell verwendet werden können.

Die Befehle ...

  • ShowAllCommandsCommand
    (zeigt die Liste aller Befehle an, Syntax: "?")
  • ShowAllCommandsWithHelpCommand
    (zeigt alle Befehle samt Hilfe an, Syntax: "??")
  • ShowCommandHelp
    (zeigt die Hilfe zu einem Befehl an, Syntax "? <befehl>"

... werden automatisch zum DefaultContext einer Shell hinzugefügt.

@see shell.Command
